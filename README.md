# 产品管理 - product-management

面向产品经理的全栈产品管理工具集。从PRD生成、用户故事拆解到竞品分析、路线图更新，覆盖产品全生命周期的核心工作流。

## 子技能列表

| 子技能 | 功能 | 触发关键词 |
|--------|------|-----------|
| PRD生成 | 输入功能需求，生成结构化PRD文档 | PRD生成, 写PRD, 需求文档, 功能规格 |
| 产品指标复盘 | 输入产品指标数据，输出健康度评估和趋势分析 | 指标复盘, 数据复盘, DAU分析 |
| 产品脑暴 | 围绕产品问题进行结构化创意发散 | 产品脑暴, 头脑风暴, brainstorm |
| 用户反馈分析 | 上传用户反馈数据，自动分类统计提取高频模式 | 用户反馈分析, 用户反馈, 用户声音 |
| 用户故事拆解 | 从Epic拆解出独立可交付的User Story | 用户故事拆解, 用户故事, story拆分 |
| 竞品分析 | 输入竞品列表，输出功能对比和SWOT分析 | 竞品分析, 竞品调研, 竞品跟踪 |
| 路线图更新 | 汇总迭代状态，评估里程碑进度 | 路线图更新, roadmap, 版本规划 |
| 需求优先级排序 | 用RICE/ICE/MoSCoW/Kano模型辅助排序 | 需求排序, 需求优先级, RICE |

## 使用方法

通过 Marvis 对话自然触发，说出需求即可自动匹配对应子技能。

## 协同技能

- 市场营销：PRD通过后调用市场营销生成配套营销文案
- 咨询交付：PRD方法论和用户研究支持咨询项目的需求分析阶段
